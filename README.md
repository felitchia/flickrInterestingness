# Flickr Interestingness


Force layout visualisation of flickr intresstingness data, following the "Understanding D3 Force Layout Tutorial" by John Alexis Guerra Gómez. 
Few changes made to the original instructions tought to fit better my purpose of learning d3.js

![Screenshot] (https://s32.postimg.org/iz981p4v9/fickrintr.jpg)

-Node size proportional to number of views
-Data organized alphabetically on the x axis and by number of views on y axis by soft constrains;
-Same colors indicate same clusters;
-Collision detection in order to avoid node overlaping;
